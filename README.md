Dotfiles

My personal macOS setup focused on keyboard-driven window management using Yabai and skhd.

Tools
	•	Yabai — tiling window manager for macOS
	•	skhd — hotkey daemon for keyboard shortcuts

Keybindings

Window Focus
	•	ctrl + h/j/k/l → move focus (left/down/up/right)

Window Resize
	•	ctrl + alt + h/j/k/l → resize windows

Window Swap
	•	shift + ctrl + h/j/k/l → swap windows

Window Warp
	•	ctrl + alt + shift + h/j/k/l → move windows in layout

Other
	•	ctrl + return → toggle float
	•	ctrl + alt + r → rebalance layout

Behavior
	•	BSP tiling layout
	•	No gaps or padding
	•	Mouse follows focus
	•	Certain apps are excluded from tiling
